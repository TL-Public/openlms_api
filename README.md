This repository contains the source code for a lightweight, scalable, and modular microservice application built using [Spring Boot](https://spring.io/projects/spring-boot). 

The application exposes apis for maintaining the master data needed for Course Content Management

---

## Technical Features

- **RESTful APIs**: Exposes robust and secure endpoints.
- **Modular Architecture**: Built with microservice best practices.
- **Database Support**: Configurable with PostgreSQL, MySQL, or MongoDB.
- **OpenAPI Documentation**: Auto-generated API documentation with Swagger.

## Application Features
- **APIs for CRUD**: Exposes robust and secure endpoints for performing CRUD operations on various business objects like Courses, Chapters, Videos, Users and other contents
- **Course Listing**
- **Training Center Listing**
- **Dashboard for course distributions across regions**
- **Dashboard on view statistics**
